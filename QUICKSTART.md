# Quick Start Guide

## Starting the CMS

**Windows:**
Simply double-click `run.bat` in the project folder, or open a command prompt and run:
```
venv\Scripts\python app.py
```

**Mac/Linux:**
```bash
venv/bin/python app.py
```

The CMS will be available at: **http://127.0.0.1:5000/**

## Your First Site in 5 Minutes

### Step 1: Create a Site
1. Open http://127.0.0.1:5000/ in your browser
2. Click "Create New Site"
3. Enter a name (e.g., "My First Site")
4. Click "Create Site"

### Step 2: Create a Page
1. Click "New Page"
2. Enter title: "Home"
3. Slug will auto-fill as "home"
4. Click "Create Page"

### Step 3: Build Your Page
1. Drag widgets from the left sidebar onto the canvas
2. Try these widgets:
   - **Heading**: Add "Welcome to My Site"
   - **Rich Text**: Add some description text
   - **Button**: Add a call-to-action button
   - **Card**: Add a feature card
3. Click the pencil icon on any widget to edit it
4. Click "Save" when done

### Step 4: Preview Your Page
Click the "Preview" button to see your page live!

### Step 5: Create a Menu
1. Go back to your site (click "Back to Site")
2. Click "New Menu"
3. Name it "Main Menu"
4. Choose position: "Top"
5. Click "Create Menu"
6. Click "Add Item"
7. Add a menu item:
   - Label: "Home"
   - Link Type: "Page"
   - Select your "Home" page
8. Check "Active" to show on site
9. Click "Save Menu"

### Step 6: Create a Footer
1. Go back to your site
2. Click "New Footer"
3. Name it "Main Footer"
4. Drag in a "Copyright" widget
5. Edit it to add your copyright text
6. Check "Active"
7. Click "Save Footer"

### Step 7: View Your Complete Site
Go to preview again - you'll now see your page with menu and footer!

## Widget Guide

### Most Popular Widgets

**Heading**
- Add H1-H6 headings
- Great for page titles and section headers

**Rich Text**
- Add formatted text content
- Supports paragraphs and simple HTML

**Button**
- Call-to-action buttons
- 6 Bootstrap styles available
- Link to pages or external URLs

**Card**
- Bootstrap card component
- Add title, text, and optional image
- Perfect for features or team members

**Columns**
- Create multi-column layouts
- Flexible width distribution
- Great for side-by-side content

**Custom HTML**
- Add any HTML code
- Use any Bootstrap component
- Full customization

## Tips & Tricks

1. **Reordering Widgets**: Just drag and drop to reorder
2. **Deleting Widgets**: Hover and click the trash icon
3. **Menu Order**: Drag menu items to reorder them
4. **Active Toggle**: Only one menu and one footer should be active per site
5. **Slugs**: Use lowercase and hyphens (e.g., about-us, contact-us)
6. **Bootstrap Classes**: All widgets use Bootstrap 5 for styling
